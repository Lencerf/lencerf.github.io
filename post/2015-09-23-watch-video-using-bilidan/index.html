Watch Video on Bilibili via bilidan.py on OS X

September 23, 2015

Install homebrew

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Install python3, ffmpeg and mpv using homebrew

brew install python3 ffmpeg Caskroom/cask/mpv

Get the python script bilidan.py

git clone https://github.com/m13253/BiliDan.git

Let bilidan.py executable

chmod u+x BiliDan/bilidan.py

Open the terminal and run the following codes:

BiliDan/bilidan.py http://www.bilibili.com/video/av715519/

Now we can watch the video without Flash Player!

Tips: you may need to wait the mpv player for some minutes the first time you run the codes above. Just be patient.

Thanks to m13253 for the projects BiliDan and danmaku2ass!

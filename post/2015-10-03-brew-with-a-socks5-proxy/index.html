brew with a socks5 proxy

October 3, 2015

Finally, I find the right way to use homebrew.

set a socks proxy with git for the http transport

git config --global http.proxy 'socks5://127.0.0.1:1081'

set a socks proxy for curl: jsut add proxy=socks5://127.0.0.1:1081 to ~/.curlrc.

Too much time have been wasted because of the poor network.
